# Rectangle Calculator

Программа для расчета параметров прямоугольника.

### Описание проекта
Данный проект представляет собой простой калькулятор на Python для вычисления:
- **Периметра** прямоугольника (\(P = 2 \times (a + b)\))
- **Площади** прямоугольника (\(S = a \times b\))
- **Диагонали** прямоугольника (\(d = \sqrt{(a^2 + b^2)}\)).

### Назначение
Программа предназначена для быстрого расчета геометрических параметров прямоугольника по заданным длинам сторон.

### Требования
- Python 3.x
- Модуль math (входит в стандартную библиотеку)

### Установка и запуск
1. Клонируйте репозиторий:
   \`\`\`bash
   git clone git@github.com:Octop186/Octop1inf.git
   cd Octop1inf
   \`\`\`
2. Запустите программу:
   \`\`\`bash
   python3 rectangle.py
   \`\`\`
